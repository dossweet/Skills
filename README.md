# 🚀 CodeGrow: 你的 AI 工程化成长引擎

> **不再让你的代码只是代码，让每一次提交都成为成长的里程碑。**

---

## 🌟 什么是 CodeGrow?

**CodeGrow** 是你的专属 AI 研发资产与成长系统。它不仅仅是一个工具，更是你的**工程化外脑**。

在 AI 辅助编程的时代，CodeGrow 帮你自动捕捉开发过程中的灵感闪光、关键决策和技术沉淀，将零散的编码工作转化为系统化的**工程资产**和可视化的**个人成长记录**。

告别繁琐的手动文档，拥抱自动化的成长闭环！

---

## ✨ 核心能力

### 1. 📝 全自动开发记录 (Capture Development Records)
自动整理需求分析、技术方案、架构决策和避坑指南。让每一行代码背后的思考都有迹可循。

### 2. 🧠 智能技能提炼 (Abstract Reusable Skills)
从日常代码中萃取通用的算法策略、设计模式和工程套路，自动生成可复用的技能卡片。**这是你带得走的核心竞争力。**

### 3. 📈 每日成长复盘 (Generate Growth Reports)
基于你的编码活动，智能分析技术复杂度与突破点，自动生成深度成长报告。
*   🚀 **支持同步飞书/Lark 云文档**，让团队看见你的进步。
*   📂 本地归档，数据完全掌控。

### 4. 🤖 AI 长期记忆 (Maintain AI Context)
为 AI 助手构建项目的“海马体”。自动摘要历史会话与决策上下文，让 AI 越用越懂你，无需重复背景。

### 5. 🛡️ 强制成长守门 (Enforce Quality via Git Hooks)
内置 Git Hooks (pre-push)，确保在代码提交前完成当天的成长复盘。**拒绝由于惰性而错失的复盘机会。**

---

## 🛠️ 如何开始？

只需在对话中召唤 CodeGrow：

> "帮我总结一下今天的工作"
> "提取这个功能的复用技能"
> "生成今天的成长报告"

CodeGrow 将会自动引导你完成：
1.  **初始化**：自动配置 `.codegrow` 目录与 Git Hooks。
2.  **分析**：回顾你的代码变更与决策。
3.  **生成**：产出记录、技能卡片与成长报告。
